🔥 Firebase 설정 필요 (1회만)
친구와 실시간 공유를 위해 무료 Firebase DB가 필요해요.
아래 순서대로 따라하세요 (5분이면 돼요!)
1. console.firebase.google.com 접속
2. 프로젝트 만들기 → 이름 입력 (예: fitlog)
3. Realtime Database → 데이터베이스 만들기
4. 테스트 모드로 시작 → 데이터베이스 URL 복사
예) https://프로젝트명-default-rtdb.firebaseio.com
